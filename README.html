Blog Backup Tool

This portable Windows folder contains everything needed to run the backup tool. No Node.js installation, browser setup, or terminal commands are required.

Start

  1. Right-click the downloaded ZIP and choose Extract All.
  2. Open the extracted BlogBackup folder.
  3. Double-click Start Backup.bat.
  4. Your browser opens http://localhost:3000.
  5. Paste a Wenxuecity blog home page URL.
  6. Leave Incremental backup checked unless you want to re-save pages already archived.
  7. Click Next.
  8. Select the categories to archive.
  9. Click Start Backup.

Keep the small black server window open while using the tool. Close it when you are done.

Output

The generated static archive is written to the archive folder. Open archive\index.html directly in a browser to view saved blogs and categories. Exported full-archive ZIP files are written to exports.

Profiles

Profiles live in app\profiles. The app detects Wenxuecity blog URLs automatically. Other profiles are still available for future site support.

Portable ZIP

For maintainers: run tools\build-portable.ps1 to create BlogBackup.zip. The build script downloads a Windows Node.js runtime, installs app dependencies, installs Playwright Chromium, and prepares empty writable archive, export, and log folders.
